Tests a char matrix for total unimodularity.

Returns true if and only if matrix is TU.

If decomposition is not NULL and the algorithm has to test regularity of the support matrix, then *decomposition will point to a decomposition tree for which the caller must use TUfreeDec to free memory. It is set to NULL otherwise.

If submatrix is not NULL and the matrix is not TU, then a submatrix with an absolute determinant larger than 1 will be searched, which may cause extra computational effort. In this case, *submatrix will point to this submatrix for which the caller must use TUfreeSubmatrix to free memory. It is set to NULL otherwise.
